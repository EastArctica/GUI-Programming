## What it does

Basic scrabble game with one row. You can drag tiles to make words and it calculates score with bonus squares.

[Put your github link here]

## Features that work
- Random tiles from the bag
- Drag and drop tiles to board
- Scoring with bonus squares (DW, DL, TL)
- Can play multiple words 
- Score adds up
- Can reset game
- Word validation with dictionary
- Blank tiles work
- 7 letter words get 50 bonus points

## How to run
Just open index.html in browser.

## Files
- index.html - main page
- assets/css/styles.css - styling
- assets/js/game.js - game code
- assets/data/pieces.json - tile data  
- assets/data/dictionary.txt - word list
- assets/images/tiles/ - tile pictures
